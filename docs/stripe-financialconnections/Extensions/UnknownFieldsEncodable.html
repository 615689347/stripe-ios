  • additionalParameters

    You can use this property to add additional fields to an API request that are not explicitly defined by the object’s interface.

    This can be useful when using beta features that haven’t been added to the Stripe SDK yet. For example, if the /v1/tokens API began to accept a beta field called “test_field”, you might do the following:

    var cardParams = PaymentMethodParams.Card()
// add card values
cardParams.additionalParameters = ["test_field": "example_value"]
PaymentsAPI.shared.createToken(withParameters: cardParams completion:...)
